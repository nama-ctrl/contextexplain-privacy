# ContextExplain PRO – Privacy Policy

Last updated: April 2026

## Overview
ContextExplain PRO is a Chrome extension that explains selected text using AI.

## Data We Collect
We do not collect any personal data. The only data stored is your OpenAI API key, which is saved locally on your device using Chrome's storage API and is never transmitted to us.

## Data We Transmit
When you select text, it is sent to our server solely to generate an explanation via the OpenAI API. We do not store, log, or share this text.

## Third Parties
We use OpenAI's API to generate explanations. Please refer to OpenAI's privacy policy for their data practices.
